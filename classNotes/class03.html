String Functions

  • .toLowerCase() - Returns value of string to lowercase
  • .toUpperCase() - Returns value of string to all uppercase
  • .substring(x,y) - Returns value between x & y of string.
  • .push() - removes & returns the last value out of the array
  • .shift() - removes & returns the first value
  • .unshift() - adds values to the beginning of an array
  • .reverse() - reverses the array order
  • .join() - combines a string, if a string is in between (' ') it will replace the comma in the string with said value.

Array Itteration

  • .forEach() - .forEach(function(passVariable){console.Log(passVariable) }); Returns output.
  • .every() - tells wether or not a each element passes by the provided function.
  • .some() - tells wether some of the elements in the array passes a function. Returns true of false.
  • .filter() - creates new array with elements that pass test - does not mutate original array.
  • .map() - creates a new array with mutated elements called by function.
