Arrays Methods
  • Square brackets define an array []
  • [].push() adds the value at the end of the arry
  • [].unshift() adds the value to the begining of the array
  • [].pop() pushes the last value in the array out of the sequence
  • [].shift() removes the first value out of the array
  • [].reverse() reverses the array values (backwards)
  • [].join() combines all elements in the array
Array Iteration
  • [].forEach() takes each value of the array, if used in a function a number can be applied to each array position. Does NOT change the original array
  • [].every() tests if every part of the array will pass a funciton array123.every(function (num) return num % 2 === 0) all array values are even. Divide by 2 with no remainders
  • [].filter() filter the array based some values
  • [].some tests if some of the values pass the function
  • [].filter() creates a new array with the elements that pass the test
  • [].map() new array with function being called on elements on this array
